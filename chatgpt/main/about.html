This application is designed for Windows users who want to communicate with this robot easily and with a simple interface, or the robot is banned in their country
For any inquiry, problem or suggestion, do not hesitate to contact us from the (Contact Us) page.

how to use

After downloading and installing the application, open it

will find the following items

  • message: To type your message that you want to send to chatgpt
  • send message: To send your message to chatgpt
  • Messages: Here are all the messages you've sent

    • note If you send a message with the same content, the old message will be replaced with the new message

  • chatgpt respons : After selecting the message from the list of messages, here is the content of the message

    • note This item is a list because the read-only edit box is not compatible with screen readers

  • respons: Copy respons to the clipboard
  • Listen: You can listen to the response
  • Settings: to open Settings

    • There is a help menu on the menu bar, which is self-explanatory

    Settings

    • app language: to select the application language
    • listen to respons language: to choose the language for listening the response

notes

To use this application you must know the following things

  • The application is open source
  • The application works with my own api key, meaning that it can stop at any time, and if it stops, please feel free to contact me
  • The application was built in the python programming language
  • The app uses Google's text-to-speech servers to listen to responses
  • If you want to add your language to the program, feel free to contact me
